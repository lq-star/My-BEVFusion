# 一、项目文件说明

---
1. configs
  基础配置文件，其他的继承它。
  
2. data
  数据集存放在这。
  
3. mmdet3d
  提供通用 3D 检测/分割模型、数据集接口和工具函数
  
4. projects
  本项目核心代码文件。
  
5. tool
  训练、测试、数据转换与日志分析等脚本工具。
  
6. requirements.txt
  Python 依赖列表，用于创建虚拟环境或一次性安装本项目所需的第三方库。
  
7. setup.cfg
  项目打包和工具配置文件。
  
8. setup.py
  安装脚本，可通过 `python setup.py develop` 或 `pip install -e .` 以开发模式安装本项目。
